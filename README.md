# 🚬 Smoking Health Risk Analysis Dashboard (Power BI)

## 📌 Overview

This project explores the **impact of smoking on human health** using interactive Power BI dashboards. It focuses on identifying **high-risk diseases and affected organs**, helping healthcare professionals take **data-driven preventive actions**.

---

## 🎯 Business Problem

Healthcare organizations face rising costs due to preventable diseases. This project helps answer:

🔍 Which diseases are strongly linked to smoking?
🫁 Which organs are most affected?
📊 How can healthcare providers prioritize screening programs?

---

## 📁 Dataset Details

The project uses multiple datasets integrated through Power BI:

📄 `health_dataset.csv` → Patient information & smoking status
📄 `condition.csv` → Disease/condition data
📄 `Organs.csv` → Organ classification
📄 `Image Dataset.csv` → Visualization mapping

---

## 🧹 Data Cleaning & Preparation

✔ Removed duplicate records
✔ Handled missing values
✔ Standardized disease & organ names
✔ Built relationships using data modeling

---

## 📊 Key Performance Indicators (KPIs)

📌 % of Smokers vs Non-Smokers
📌 Disease prevalence by smoking status
📌 Organ-wise disease impact
📌 High-risk condition concentration

---

## 📈 Dashboard Features

✨ Interactive filters & slicers
✨ Dynamic visuals (bar charts, pie charts)
✨ Organ-wise and disease-wise breakdown
✨ Easy-to-understand storytelling visuals

---

## 🔍 Key Insights

🚨 Smokers show a **high concentration of lung diseases**
❤️ Strong correlation with **cardiovascular conditions**
🧬 Chronic smokers often have **multi-organ involvement**
📉 Non-smokers show significantly **lower disease occurrence**

---

## 💡 Business Recommendations

✅ Prioritize **respiratory screening** for smokers
✅ Promote **smoking cessation programs**
✅ Allocate resources to **cardio & pulmonary care**
✅ Use analytics for **early risk detection**
